# Cambios de panel de resultados

- La salida principal ya no se muestra como consola negra.
- Durante la revisión se muestra un panel amigable de progreso.
- Al terminar se muestra un informe visual con:
  - Resultado final.
  - Tipo de revisión.
  - Pines disponibles en BSDL.
  - Pines usados en esa prueba.
  - Tabla de pines revisados con color: verde PASS, rojo FAIL/ERROR.
  - Tabla separada para conexiones tipo TX/RX/SPI/I2C/GPIO.
- Se mantiene botón para copiar todo.
- Se agregó botón para ver/ocultar log completo por si se necesita depurar.
- La lista lateral de pines vuelve a mostrar texto corto del BSDL al lado de cada pin, en pequeño.
